# deku-redux-example

> An example of Deku Redux async application

## Install

```sh
git clone https://github.com/andrepolischuk/deku-redux-example
cd deku-redux-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
